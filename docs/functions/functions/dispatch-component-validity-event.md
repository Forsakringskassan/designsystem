Dispatch an ComponentValidityEvent to the given element.

```ts
export function dispatchComponentValidityEvent(
    element: Element,
    detail: ComponentValidityEvent,
): void;
```

-   `element` - the element to dispatch event to
-   `detail` - the event detail
