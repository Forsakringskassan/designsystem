Returnerar objekt med slutgiltiga mappningar mellan validatorer och feltexter för användning med `ValidationService.setErrorMessages()`.

## Syntax

```ts nocompile nolint
build();
```

### Returvärde

Ett objekt med mappning mellan validatorer och feltexter.
