Returnerar objekt med slutgiltiga mappningar mellan validatorer och feltexter för användning med `ValidationService.addValidationErrorMessages()`.

## Syntax

```ts nocompile nolint
build();
```

### Returvärde

Ett objekt med mappning mellan validatorer och feltexter.
