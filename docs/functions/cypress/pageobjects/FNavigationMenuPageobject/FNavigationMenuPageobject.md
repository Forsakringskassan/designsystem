Represents `f-navigation-menu` with access to its elements and `i-popup-menu` page object.

## Syntax

```ts nocompile nolint
new FNavigationMenuPageobject(selector?);
```

### Parametrar

`selector: string` {@optional}
: the root of the navigation menu, usually `.imenu`.
