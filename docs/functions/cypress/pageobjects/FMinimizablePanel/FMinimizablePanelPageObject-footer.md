Hämtar ut det användardefinierade innehållet från `footer`-slotten i {@link FMinimizablePanel Minimerbar panel} (FMinimizablePanel).

## Syntax

```ts nocompile nolint
footer();
```

### Returvärde

`HTMLElement` med det användardefinierade innehållet från `footer`-slotten.
