Hämtar ut den fördefinierade öppna/minimera-knappen i {@link FMinimizablePanel Minimerbar panel} (FMinimizablePanel).

## Syntax

```ts nocompile nolint
toggleButton();
```

### Returvärde

`HTMLButtonElement` med den fördefinierade öppna/minimera-knappen.
