Hämtar ut det användardefinierade innehållet från `header`-slotten i {@link FMinimizablePanel Minimerbar panel} (FMinimizablePanel).

## Syntax

```ts nocompile nolint
header();
```

### Returvärde

`HTMLElement` med det användardefinierade innehållet från `header`-slotten.
