Hämtar ut det användardefinierade innehållet från `content`-slotten i {@link FMinimizablePanel Minimerbar panel} (FMinimizablePanel).

## Syntax

```ts nocompile nolint
content();
```

### Returvärde

`HTMLElement` med det användardefinierade innehållet från `content`-slotten.
