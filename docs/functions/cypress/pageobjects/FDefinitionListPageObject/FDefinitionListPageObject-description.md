Används för att komma åt beskrivningen för en definition i definitionslistan.

## Syntax

```ts nocompile nolint
description(index);
```

### Parametrar

`index: number`
: Index för definitionen i definitionslistan.

### Returvärde

`HTMLElement` som innehåller definitionens beskrivning.
