Används för att komma åt termen för en definition i definitionslistan.

## Syntax

```ts nocompile nolint
term(index);
```

### Parametrar

`index: number`
: Index för definitionen i definitionslistan.

### Returvärde

`HTMLElement` som innehåller definitionens term.
