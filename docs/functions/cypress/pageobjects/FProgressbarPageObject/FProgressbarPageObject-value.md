Används för att hämta det nuvarande värdet på försloppsindikatorn (0-100).

## Syntax

```ts name=syntax nocompile nolint
value();
```

### Returvärde

`number` som innehåller värdet som förloppsindikatorn har just nu.

FProgressbarPageObject-value.vue
```

```
