Detaljpanel (FDetailsPanel) används tillsammand med {@link FPageLayout Applikationslayout} för att visa detaljer om ett av användaren valt objekt.

```import
FDetailsPanelExample.vue
```

## Användning

## Flera detaljpaneler i samma yta

exclusive

## Anpassa texter

Komponenten använder följande texter:

| Textnyckel                 | Standardtext | Beskrivning                                |
| -------------------------- | ------------ | ------------------------------------------ |
| `fkui.details-panel.close` | Stäng        | Skärmläsartext för stäng-krysset i header. |
