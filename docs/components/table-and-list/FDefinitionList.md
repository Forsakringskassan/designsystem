Definitionslista används för att visa en lista av definitioner.

```import live-example
FDefinitionListLiveExample.vue
```

## Justerad layout

Justerad layout aktiveras med attributet `aligned` och rekommenderas om termerna för samtliga definitioner är ungefär lika breda.

## Användning av komponent

**Komponent motsvarar följande HTML element:** [Description List](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/dt)
